Section 1 — Boolean Attribute Edge Cases

HTML boolean attributes: presence = true, regardless of value. disabled="false" is STILL disabled.

disabled (no value)

Expected: is disabled = true

disabled="disabled"

HTML: disabled="disabled"
Expected: is disabled = true

disabled="false" (STILL disabled!)

HTML: disabled="false"
⚠️ In HTML, presence = true. This IS disabled despite value "false"

disabled="" (empty string)

HTML: disabled=""
Expected: is disabled = true (empty string = present)

Not disabled (control)

No disabled attribute
Expected: is enabled = true

aria-disabled="true" (not natively disabled)

HTML: aria-disabled="true" but NO native disabled
⚠️ Is this "disabled" to the assertion engine?

Decimal Value Element

Boundary Element
